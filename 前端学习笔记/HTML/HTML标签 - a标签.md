# HTML标签 - a标签

什么是a标签？

+ a标签的作用：就是用于控制页面与页面之间的天转
+ a标签的格式：`<a href="指定需要跳转的页面地址"> 需要展现给用户查看的内容</a>`

a标签的属性：

+ target：专门用于控制如何跳转
  + _self：用于在当前选项卡中跳转，也就是不新建界面跳转。默认就是`_self`。
  + _blank：用于在新的选项卡中跳转，也就是新建界面跳转。
+ title：同img标签的title一样，都是用于显示鼠标悬停时展示的提示。

注意点：

+ a标签不仅可以让文字可以点击，还可以让图片可以点击。
+ 一个a标签必须有一个href属性，否则a标签不知道要跳转到什么地方。
+ 如果通过a标签的href属性置顶一个url地址，必须在地址前面加上http://或者https://。

+ a标签的href除了可以指定一个网络地址以外，还可以指定一个本地地址。

# HTML假链接

什么是假链接？

点击之后不会跳转的链接就是假链接。

假链接存在的意义：在开发前期，其他页面都还没有写出来，我们不知道到底需要跳转到什么地方，我们就需要假链接代替，当到了项目后期，其他页面完成后，将假链接替换成真链接。

假链接的格式：`<a href="#"></a>`或者`<a href="javascript:"></a>`

假链接两种格式区别：#会自动回到网页顶部，但是javascript:不会回到网页顶部。

# HTML锚点

+ 要想通过a标签跳转到指定位置，必须告诉a标签一个独一无二的身份证号码，这样a标签才能在当前界面中找到需要跳转到的目标位置。
+ 如何给HTML中的标签绑定一个独一无二的身份证？
  + 在HTML中，每个标签有一个名称叫做id的属性。
  + 这个属性就是用来给标签指定一个独一无二的身份证号码
+ 所以想要实现通过a标签跳转到指定的位置分为两步：
  + 给目标位置的标签添加一个id属性，然后指定一个独一无二的值
  + 告诉a标签你需要跳转到的目标标签对应的独一无二的身份证号码是多少
+ 格式：`<a href="#id名称"></a>`或者`<a href="目标路径#id名称"></a>`
+ 注意点：
  + 通过a标签跳转到指定位置是没有过渡动画的，是直接跳转到指定位置。
  + a标签除了可以跳转到当前界面指定位置外，还可以直接跳转到其他界面的指定位置。
