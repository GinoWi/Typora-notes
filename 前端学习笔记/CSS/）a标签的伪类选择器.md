### a标签的伪类选择器

通过我们的观察发现a标签存在一定的状态

+ 默认状态，从未被访问过
+ 被访问过的状态
+ 鼠标长按的状态
+ 鼠标悬停时的状态

作用：专门用来修改a标签不同状态下的样式的。

格式：

```
:link 从未被访问过
:visited 被访问过的状态
:hover 鼠标悬停时的状态
:active 鼠标长按状态
```

注意点：

+ a标签的伪类选择器可以单独的出现，也可以一起出现。
+ a标签的伪类选择器如果一起出现，有严格的顺序要求（默认状态 -》 被访问过 -〉鼠标长按 -》 鼠标悬停）
+ 如果默认状态的样式和被访问过状态的样式一样，那么可以缩写。可以简化使用a标签的color属性



在开发过程中编写a标签的伪类选择器最好能写在标签选择器的后面。

在开发过程中和a标签盒子相关的属性都写在标签选择器中（显示模式/高度/宽度……）。

在开发过程中和a标签文字/背景相关的都写在伪类选择器中。
