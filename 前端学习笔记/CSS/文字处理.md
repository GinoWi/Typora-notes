### 行高

+ 什么是行高？

  + 在CSS中所有的行都具有行高属性

+ 格式：

  ```
  line-height: 80px;
  ```

+ 注意点：

  + 行高和盒子的高度不是同一个概念，行高是指的这一行内容的高度，盒子的高度指的是标签的高度。

+ 规律：

  + 文字在行高中默认是垂直居中的。
  + 在开发过程中，经常将盒子的高度和行高设置为一样，这样就能保证一行文字在盒子的高度中是垂直居中的，简而言之，要想一行文字在盒子中垂直居中，那么只需要设置这行文字的“行高等于盒子的高”即可。
  + 在开发过程中，如果一个盒子有多行文字，那么我们就不能使用设置行高等于盒子的高度来实现让文字垂直居中，只能通过设置padding来让文字居中。



### 字号

+ 注意点：
  + 在开发中，如果一个盒子中存储的是文字，一般情况下，我们会以盒子左边的内边距作为基准，不会以右边的内边距作为基准，因为右边内边距会有误差。右边内边距来源：因为右边如果剩余空间放不下一个文字，就会换行展示，所以右边内边距就会产生误差。
  + 顶部的内边距并不是边框到文字底部的距离，而是边框到行高顶部的距离。
