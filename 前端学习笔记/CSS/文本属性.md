# 文本属性

一、文本装饰属性

格式：`text-decoration:underline;`

取值：

underline:下划线

line through：删除线

overline：上划线

none：什么都没有，最常见的用途就是用于去掉超链接的下划线

快捷键：

td + tab键：`text-decoration: none;`

tdu + tab : `text-decoration: underline;`

tdl + tab : `text-decoration: line-through;`

tdo + tab : `text-decoration: overline;` 



二、文本水平对齐的属性

格式：`text-align: center;`

取值：

left：左

right：右

center：中

快捷键：

ta + tab ：`text-align: left;`

tar+tab : `text-align: right;`

tac+tab : `text-align: center;`



三、文本缩进的属性

格式：`text-indent: 2em;`

取值：

2em，其中em是单位，一个em代表缩进一个文字的宽度。

10px，也可以用像素作为单位

快捷键：

ti：`text-indent: ;`

ti2e : `text-indent: 2em;`



四、修改文本颜色属性

在CSS中如何通过color属性修改文字颜色：

格式：`color: 值;`

取值：

1. 通过英文单词赋值
   + 一般情况下，常见的颜色都有对应的英文单词，英文单词能够表达的颜色是有限制的。也就是说不是所有的颜色都能通过英文单词来做表达。
2. 通过rgb赋值
   + rgb其实就是三原色，r-red，g-green，b-blue
   + rgb(红色, 绿色, 蓝色)
   + 那么这个格式中的第一个数字就是用来设置三原色的光源原件红色显示的亮度
   + 第二个数字设置三原色的光源原件绿色显示的亮度
   + 第二个数字设置三原色的光源原件蓝色显示的亮度
   + 这其中的每一个数字她的取值是0-255之间，0代表不发光，255代表发光，值越大就越亮
   + 只要让红色、绿色、蓝色的值都一样就是灰色，而且这个值越小越偏向黑色，值越大越偏向白色。
3. 通过rgba赋值
   + CSS3推出
   + rgba中的rgb和前面讲解的一样，只不过多了一个a
   + 那么这个a代表透明度，取值0-1，取值越小代表越透明。
4. 通过十六进制赋值
   + 在前端开发中，通过16进制来表示颜色， 其实本质就是RGB，十六进制中是通过每两位表示一个颜色。
   + 例如：`#FFEE00` FF表示R， EE表示G，00表示B
5. 通过十六进制缩写赋值
   + 在CSS中只要十六进制的颜色每两位的值都是一样的就可以简写为一位。
   + 例如：`#FFEE00`  ==> `#FE0`

注意点：

1. 如果当前颜色对应的两位数字不一样，就不能简写。
2. 如果两位相同的数字不是属于同一个颜色的，也不能简写。
