`text-align` 属性能直接让 `div` 中的 `span` 元素水平居中，其核心原因在于 ‌**`text-align` 的作用对象是“块级元素内部的行内内容”**‌，而非元素自身。

具体来说，其工作原理如下：

- `div` 是一个‌**块级元素**‌，它默认会占据一整行的宽度。
- `span` 是一个‌**行内元素**‌，它不会独占一行，而是像文字一样嵌套在块级元素内部。
- `text-align: center` 这个属性，‌**只能应用于块级元素**‌，它的作用是‌**将其内部的所有行内内容（包括文本、`span`、`img`、`a` 等）在水平方向上居中对齐**‌。

因此，当你在 `div` 上设置 `text-align: center` 时：

1. `div` 作为块级容器，接收了这个样式。
2. `div` 内部的 `span` 元素被识别为“行内内容”。
3. 浏览器将 `span` 及其包含的文字，在 `div` 的宽度范围内进行水平居中排列。

这并不是 `span` 自己“居中”了，而是 `span` 作为 `div` 的内部内容，被 `div`的 `text-align` 样式所控制。

### 关键要点总结

- ‌**`text-align` 作用于父容器**‌：必须将 `text-align: center` 应用在 `span`的直接父级（如 `div`）上，而不是 `span` 本身。
- ‌**只影响行内内容**‌：`text-align` 只能居中行内元素或行内块元素（如 `span`、`img`），不能直接居中一个块级元素（如另一个 `div`）。
- ‌**与 `span` 的默认行为无关**‌：`span` 默认是行内元素，它本身没有宽度概念，`text-align` 的效果是父容器对其内部内容的排布结果。

简单来说，`div` 是一个“盒子”，`span` 是盒子里的“小纸条”。`text-align: center` 是告诉这个“盒子”：“把里面的所有小纸条都摆到盒子中间”。‌
