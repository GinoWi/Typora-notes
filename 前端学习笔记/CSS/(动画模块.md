# 动画模块

过渡和动画的不同点：

+ 过渡必须认为出发才会执行动画，动画不需要人为触发。

过渡和动画的相同点：

+ 过渡和动画都是用来给元素添加动画的。
+ 过渡和动画都是系统新增的一些属性。
+ 过渡和动画都需要满足三要素才会有动画效果。

动画三要素：

+ 告诉系统需要执行哪些动画，取值就是执行动画的动画名称
+ 告诉系统我们须啊摇自己创建一个名称叫做某某的动画
+ 告诉系统动画持续时间有多久

其他属性：



动画模块连写格式：

animation: 动画名称 动画时长 动画运动速度 延迟时间 执行次数 是否需要往返动画;

动画模块连写格式简写：

animation: 动画名称 动画时长;
