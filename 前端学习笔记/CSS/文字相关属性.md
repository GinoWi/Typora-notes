# 文字相关属性

+ 规定文字样式属性
  + 格式：`font-style: italic;`
  + 取值：
    + normal：正常的，默认取值
    + italic：倾斜
  + 快捷键：
    + fs + tab键：`font-style: italic;`
    + fsn + tab键：`font-style: normal;`
+ 规定文字粗细属性
  + 格式：`font-weight:bold;`
  + 取值：
    + 单词取值：
      + bold：加粗
      + bolder：比加粗还要粗
      + lighter：细线，默认就是细线
    + 数值取值：100～900之间整百的数字
  + 快捷键：
    + fw + tab键：`font-weight: ;`
    + fwb + tab键：`font-weight: bold;`
    + fwl + tab键：`font-weight: lighter;`
    + fwbr + tab键：`font-weight: bolder;`
+ 规定文字大小属性
  + 格式：`font-size: 30px;`
  + 单位：px（pixel） 像素
  + 注意点：通过font-size设置大小一定要带单位，也就是一定要写px
  + 快捷键：fz + tab键：`font-size: ;`
+ 规定文字字体属性
  + 格式：`font-family: ;`
  + 注意点：
    + 如果取值是中文，需要用双引号或者单引号包裹。
    + 设置的字体必须是用户电脑已经安装的字体。
  + 快捷键：ff + tab键：`font-family: ;`



补充内容：

+ 如果设置字体不存在，那么系统会使用默认的字体来显示（宋体）
+ 如果设置的字体不存在，而我们又不想用默认的字体来显示怎么办？
  + 可以字体设置备选方案，格式：`font-family: "字体1", "备选方案1", "备选方案2"...;`
+ 如果想给中文和英文分别单独设置字体，怎么办？
  + 但凡是中文字体，里面都包含了英文，但凡是英文字体里面没有包含中文。也就是说中文字体可以处理英文，英文字体处理不了中文。
  + 格式：`font-family: "英文字体", "中文字体";`
  + 如果想给界面中的英文单独设置字体，英文字体就必须写在中文字体的前面
+ 在企业开发中，最常见的字体有以下几个
  + 中文：宋体、黑体、微软雅黑
  + 英文：Times New Roman/Arial
+ 并不是名称是英文，就一定是英文字体，因为中文字体都有自己的英文名称，所以是不是中文字体，主要是看能不能处理中文。比如：宋体 -》SimSun、黑体 -〉SimHei、微软雅黑 -》Microsoft YaHei



文字属性简写：

+ 简写格式：`font: {style} {weight} {size} {family};` 例如：`font: italic bold 10px "楷体";`

+ 在这种缩写格式中，有的属性值可以省略，weight可以省略。
+ 在这种缩写格式中，style和weight位置可以交换。
+ 在这种缩写格式中，所有的属性值是不可以省略的，size不能省略，family不能省略。
+ size和family属性的位置是不能随便乱放的，size一定要写在family前面，size和family必须写在所有属性的最后。
