# Python中的函数

函数是带名字的代码块，用于完成具体的工作。如果要执行函数定义的特定任务，可以调用函数完成。

接下来完成一个简单的函数定义：

```python
def greet_user():
    """显示简单的问候语"""
    print("Hello!")

greet_user()
```

需要关注函数的以下几点：

1. `def`是用来定义函数的关键字。
2. 第二行双引号引起的内容是称为**文档字符串**的注释（可以包含多行），描述了这个函数做了什么。
3. `print("Hello!")`是函数执行的内容。



参数一般情况下不需要默认值，一旦某个参数设置了默认值，那么后续的所有参数都需要设置默认值。
