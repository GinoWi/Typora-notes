# Python中的文件

### 文件基本操作

+ 打开文件

  当我们需要通过`python`实现文件打开的时候，可以使用`open`方法。

  ```python
  open(file, mode='r', buffering=-1, encoding=None, erros=None, newline=None, closefd=True, opener=None)
  ```

  参数说明：

  | 参数      | 说明                                                         |
  | --------- | ------------------------------------------------------------ |
  | file      | 文件路径名称                                                 |
  | mode      | 打开方式<br />r：只读模式，打开后不能执行写操作<br />w：只写模式，文件存在被清空，打开后不能执行读操作<br />x：创建新文件，以只写的方式打开，若文件存在则报错<br />a：追加模式，所有的写入操作都会自动地被添加到文件的末尾，而不会覆盖现有内容。如果文件不存在，Python会在当前目录下创建这个文件。<br />x+<br />a+<br />r+：读写方式打开。通过这种方式进行读写文件的时候，采用的是追加模式，如果想在原有文件内容后面追加新的内容需要将原有文件内容先读出来，然后再写；否则会从文件开始覆盖<br />w+：读写方式打开，文件存在被清空<br />二进制方式：rb，wb，xb，rb+，wb+ 场景：图片，二进制文件等 |
  | buffering | 缓存机制<br />1.表示使用缓存机制<br />-1.表示使用系统默认<br />0.表示不适用缓存机制，值对Binary有效 |
  | encoding  | 编码格式                                                     |
  | newline   | 换行符                                                       |

os模块：

相关模块：

```python
import os
import shutil
```

| os模块相关函数                        | 说明                                         |
| ------------------------------------- | -------------------------------------------- |
| os.getcwd()                           | 返回表示当前工作目录                         |
| os.mkdir(name)/os.rmdir(name)         | 创建目录/删除目录                            |
| os.makedirs(name)/os.removedirs(name) | 创建目录树/删除目录                          |
| os.listdir(path=None)                 | 获取指定目录下所有文件目录                   |
| os.rename(src, dst**, ……)             | 文件目录重命名                               |
| os.renames(old, new)                  | 递归方式重命名目录或者文件                   |
| os.walk(top, topdown=True, ……)        | 获取指定目录下所有文件目录，返回目录树迭代器 |
| os.remove(path, dir_id=None           | 删除文件                                     |

| shutil模块相关函数                              | 说明                                             |
| ----------------------------------------------- | ------------------------------------------------ |
| shutil.move(src, dst)                           | 移动目录                                         |
| shutil.copyfileobj(src, dest, length=16*1024)   | 将src内容拷贝到dest， src，dest为文件读写对象    |
| shutil.copyfile(src,dst)                        | 整个文件拷贝                                     |
| shutil.copymode(src, dst)                       | 仅拷贝全县，内容、组，用户均不变                 |
| shutil.copystat(src,dst)                        | 仅复制所有的状态信息，包括全县，组，用户，时间等 |
| Shutil.copytree(src, dest)                      | 递归方式拷贝目录树                               |
| shutil.rmtree(path[, ignore_errors[, onerror]]) | 递归方式删除目录与文件                           |

| 字段     | 说明                    |
| -------- | ----------------------- |
| st_mode  | inode保护模式           |
| st_ino   | inode节点号             |
| St_dev   | inode主流的设备         |
| st_nlink | inode的链接数           |
| St_uid   | 所有者的用户ID          |
| st_gid   | 所有者的组ID            |
| St_size  | 以字节为单位的大小      |
| St_atime | 上次访问的时间          |
| st_mtime | 最后一次修改的时间      |
| st_ctime | 由操作系统报告的“ctime” |
