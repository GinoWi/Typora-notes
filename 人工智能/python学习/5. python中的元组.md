# Python中的元组

当需要创建一系列不可修改的元素的时候，可以考虑使用Python中的元组。

元组的定义的时候一般都是使用圆括号，而不是方括号。

```python
dimensions = (200, 50)
print(dimensions[0])
print(dimensions[1])
```

## 元组的相关操作：

+ 遍历：

  同列表一样可以通过`for`循环实现元组的遍历。

  ```python
  dimensions = (200, 50)
  for dimension in dimensions
  	print(dimension)
  ```

  执行结果：

  <img src="https://picgo-storage-typora.oss-cn-shanghai.aliyuncs.com/Python-tuple-operation-1.png" alt="image-20260316151344799" style="zoom:50%;" />

+ 修改

  元组中的元素无法进行修改，但是可以给重新定义该元组变量。

  ```python
  dimensions = (200, 50)
  print("Original dimensions:")
  for dimension in dimensions:
      print(dimension)
  
  dimensions = (400, 70)
  print("New dimensions:")
  for dimension in dimensions:
      print(dimension)
  ```

  

  执行结果：

  <img src="https://picgo-storage-typora.oss-cn-shanghai.aliyuncs.com/Python-tuple-operation-2.png" alt="image-20260316151824666" style="zoom:50%;" />



补充：

元组的定义方式还可以使用关键字`tuple`

```python
tuple("1234")
tuple([1, 2, 3, 4])
```

元组常用的方法：

T.count(value) 统计value再L中出现次数

T.index(value, [start, [stop]]) 返回value第一次出现的位置，不存在产生异常
